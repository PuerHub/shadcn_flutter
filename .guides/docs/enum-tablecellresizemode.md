description: "Reference for TableCellResizeMode"

```dart
enum TableCellResizeMode {
  /// The cell size will expand when resized
  expand,
  /// The cell size will expand when resized, but the other cells will shrink
  reallocate,
  /// Disables resizing
  none,
}
```
