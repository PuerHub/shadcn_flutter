```dart
/// The direction for sorting.
enum SortDirection {
  /// No sorting applied.
  none,
  /// Sort in ascending order (A to Z, 0 to 9).
  ascending,
  /// Sort in descending order (Z to A, 9 to 0).
  descending,
}
```
